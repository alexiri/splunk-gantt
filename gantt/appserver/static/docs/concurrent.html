When there are concurrent tasks in the same category, they will be drawn underneath each other.

If you have many categories or many concurrent tasks, it may be a good idea to specify compact: "true" in your dashboard. This compressed the chart by making the bars thinner, which helps to fit more in the view.

This dashboard demonstrates several different ways of using the Gantt chart. Note, for example, the usage of base searches new to Splunk 6.2. This dashboard also shows the usage of tokenName and tokenField to make the chart emit a token instead of drilldown events.
